# 高中物理圆周运动仿真实验平台

一个基于现代Web技术构建的高中物理圆周运动教学平台，提供交互式仿真、可视化学习和智能评测功能。

## 🌟 项目特色

### 📚 三维一体学习模式
- **理论展示**：清晰的知识点梳理和公式推导
- **交互仿真**：实时参数调节和物理现象可视化
- **智能评测**：多样化题型和自适应评分系统

### 🎮 核心功能模块

#### 1. 主页面 - 理论+演示
- ✅ 圆周运动知识点系统展示
- ✅ 交互式Canvas动画演示
- ✅ 向心力矢量可视化
- ✅ PhET专业仿真架构

#### 2. 仿真页面 - 深度实验
- ✅ 完整参数控制系统（质量、角速度、拉力、半径）
- ✅ 实时物理仿真和轨迹绘制
- ✅ 绳子断裂临界条件模拟
- ✅ 三个物理矢量可视化
  - 🔵 角速度箭头（绿色）- 垂直运动平面
  - 🟡 线速度箭头（黄色）- 切线方向
  - 🔴 向心加速度（红色）- 指向圆心
- ✅ 运行方向切换（顺时针/逆时针）
- ✅ 暂停/继续和完整重置功能

#### 3. 习题系统 - 智能评测
- ✅ **3个选择题**：临界问题、公式记忆、实际应用
- ✅ **3个计算题**：含数值输入和文件上传功能
- ✅ **1个开放题**：启发性思考和拓展分析
- ✅ 智能评分系统和个性化学习建议

## 🛠 技术栈

### 前端技术
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **路由**：React Router v6
- **状态管理**：React Hooks + Context

### 图形与仿真
- **渲染引擎**：HTML5 Canvas 2D API
- **物理模型**：自定义物理引擎
- **动画系统**：requestAnimationFrame
- **架构模式**：PhET SceneryStack (MVC)

### 部署与运维
- **部署平台**：Vercel
- **版本控制**：Git
- **CI/CD**：自动化部署流程

## 📊 物理模型

### 核心公式
```
向心力：F = mω²r = mv²/r
线速度：v = ωr
角速度：ω = 2π/T
```

### 仿真特性
- **单位系统**：物理单位（米、秒、弧度）与屏幕坐标转换
- **时间缩放**：0.016倍时间缩放，确保动画观察效果
- **临界条件**：自动检测绳子断裂条件
- **轨迹记录**：实时收集运动轨迹点

## 🚀 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署到Vercel
1. Fork 本项目到您的GitHub账号
2. 登录 [Vercel](https://vercel.com)
3. 导入GitHub仓库
4. 一键部署，自动获得 `.vercel.app` 域名

4. **性能优化**：requestAnimationFrame + 轨迹点数量控制
